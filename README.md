🔐 Random Password Generator

A simple and secure Random Password Generator that creates strong passwords instantly.
This project helps users generate safe passwords with customizable options such as length and character types.
<hr>

🚀 Features
* Generate strong and secure passwords
* Custom password length
* Includes:
    * Uppercase letters
    * Lowercase letters
    * Numbers
    * Special characters
* Fast and lightweight
* Easy to use interface
<hr>

🛠️ Tech Stack
Language: JavaScript

⚙️ How It Works
The application randomly selects characters from predefined character sets and combines them to create a secure password based on user-selected criteria.

💡 Usage
1. Choose password length.
2. Select character types.
3. Click Generate Password.
4. Copy and use your secure password.

🔒 Why Strong Passwords Matter
Strong passwords help protect accounts from brute-force attacks, hacking attempts, and unauthorized access.
